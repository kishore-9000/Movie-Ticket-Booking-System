# 🎬 CinePass - Movie Ticket Booking System

A modern, full-stack movie ticket booking platform with a dark cinematic UI.

## 🚀 Features
- **Dark Cinematic UI**: Premium dark mode experience with glassmorphism.
- **Interactive Seat Selection**: Visual grid for selecting theatre seats.
- **Full Stack Architecture**: React frontend, Node.js/Express backend, and MySQL database.
- **Containerized**: Ready for deployment with Docker.

## 🛠️ Tech Stack
- **Frontend**: React 18, React Router, Vanilla CSS
- **Backend**: Node.js, Express.js
- **Database**: MySQL 8.0
- **DevOps**: Docker, Docker Compose

## 📦 Getting Started

### 1. Standalone Preview
Open `index.html` in your browser for an immediate look at the cinematic UI and seat selection prototype.

### 2. Full Stack Setup (Docker)
```bash
docker-compose up -d
```

### 3. Manual Setup
**Backend:**
```bash
cd server
npm install
npm start
```

**Frontend:**
```bash
cd client
npm install
npm start
```

## 📂 Project Structure
- `client/`: React application.
- `server/`: Express API server.
- `database/`: SQL schema and sample data.
- `index.html`: Standalone high-fidelity prototype.
