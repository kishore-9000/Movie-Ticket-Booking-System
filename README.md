# Movie Ticket Booking System

A modern full-stack movie ticket booking platform with dark cinematic UI, seat selection, QR ticket generation, and gift card support.

## Tech Stack
- Frontend: React + Bootstrap
- Backend: Node.js + Express
- Database: MySQL
- External API: TMDb API

## Features
- Movie listing with filters
- Theatre and showtime management
- Interactive seat booking
- QR-based ticket confirmation
- Gift card system
- User authentication

## Testing
- UI filter validation
- Double booking prevention
- Secure payment and gift flow
